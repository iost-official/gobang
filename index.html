How to play

1. Input your IOST account and base58-encoded seckey to wallet settings

2.a As a host, input opponent in Game Settings, and click Create Game. Then the game starts and game id appears

2.b As a guest, input the game id and click Enter Game twice (it's a bug to be fixed)

3. play with your opponent

Notice

1. There are bugs when you play with yourself, so opponent should be different from your account

2. Please report bugs in github issues
